AbstractSelector()Constructor
public function AbstractSelector(self:AbstractSelector)

Selector is a special object that contains the names and values of properties another class can use to configure itself off of. Unlike a regular Object, the Selector instance keeps an ordered list to allow ordered looping through it's values. This assures that the order of the values set will be correctly returned in the order expected.

public function merge(selector:ISelector):void

Merges the properties of one group of properties with the current instances values. The supplied properties have a higher importance and will override any properties with the same name.

It is important to note that the selectorName will also be overrode by the new properties.
